SQL - Data Exploration

This project focuses on exploring and analyzing global COVID-19 data using SQL. It leverages various advanced SQL techniques, such as Joins, CTEs, Temporary Tables, and Views. The analysis involves calculating COVID-19 infection and death percentages, tracking vaccination progress, and showcases skills in data aggregation, partitioning, and the use of functions like Cast/Convert, Group By, and Order By for deeper insights into the pandemic's global impact.

Tableau

This Tableau project analyzes global COVID-19 data through four key visualizations: a global infection map showing countries with the highest infection rates by population, a time-based graph tracking the growth of COVID-19 cases by location, a bar chart comparing total deaths across continents, and a summary chart highlighting global totals of new cases, deaths, and death percentages. The project provides clear insights into the pandemic's worldwide impact.

SQL - Data Cleaning

This project involves cleaning and optimizing a real estate dataset using SQL. Key tasks include standardizing date formats, filling in missing property addresses using related data, breaking down address fields into separate columns, standardizing binary columns, removing duplicate records, and dropping unused columns. The project demonstrates strong skills in data transformation and preparation to ensure data accuracy and consistency.

Power BI

This Power BI project provides a comprehensive breakdown of data professionals in 2022, visualizing key metrics such as average salary by job title, work-life balance happiness, salary satisfaction, and a breakdown by country. It also displays favorite programming languages and job role distribution. The report features dynamic visuals including bar charts, a donut chart, and gauges, offering insights into the demographics and preferences of survey respondents, who are primarily data professionals.

Excel

This Excel dashboard utilizes pivot tables and charts to analyze bike sales data. It includes filters for marital status, region, and education, offering insights into key metrics like average income by gender, bike purchases by age group, and customer commute distance. The combination of pivot tables and visual graphs provides a dynamic, interactive way to explore purchasing trends across different demographics.
